# midi pi

Easy-to-use midi master device build with Raspberry pi

## Requirements

### Hardware

- Raspberry pi 4
- Midi output interface device
- LCD display
- additional keyboard

### Software

- Raspbian
- node.js
- yarn
- pm2
- chrome
